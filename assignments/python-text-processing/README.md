# 📘 Assignment: Python Text Processing

## 🎯 Objective

Practice working with strings, file I/O, and text manipulation in Python by implementing functions to read, process, and analyze text data.

## 📝 Tasks

### 🛠️ Read and Display Text File

#### Description
Write a function that reads a text file and displays its contents, along with basic statistics like word count and line count.

#### Requirements
Completed program should:

- Read a text file specified by the user
- Display the full contents of the file
- Print the total number of words and lines
- Handle file not found errors gracefully

### 🛠️ String Manipulation Functions

#### Description
Create functions to perform common string operations like counting vowels, reversing text, and finding substrings.

#### Requirements
Completed program should:

- Include a function to count vowels in a given string
- Include a function to reverse the order of words in a sentence
- Include a function to find and replace all occurrences of a substring
- Provide example usage for each function

### 🛠️ Text Analysis

#### Description
Implement a simple text analyzer that provides insights about the text, such as most frequent words and average word length.

#### Requirements
Completed program should:

- Read text input from the user or a file
- Calculate and display the most frequent words (top 5)
- Calculate and display the average word length
- Ignore case sensitivity and punctuation for accurate analysis
